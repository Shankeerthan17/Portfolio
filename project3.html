Scalable Enterprise Network Design for a Two-Floor Office Environment

Project Info

The network was designed using routers, a core switch, and multiple access switches to ensure efficient data flow and segmentation. Structured cabling was implemented using CAT6 UTP cables, patch panels, and wall outlets to maintain an organized and reliable physical infrastructure.


Wireless connectivity was achieved using Wi-Fi 6 access points, supported by PoE-enabled managed switches for better performance and simplified power management. A centralized 22U rack system was designed for proper equipment organization and maintenance. FTTH (1 Gbps) internet connectivity was integrated to ensure high-speed and stable network access.
